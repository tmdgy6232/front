자바스크립트의 객체

        소프트웨어에 대한 객체지향적 방햫ㅇ에서 보면 프로그램과 객체(데이터 + 함수) 로 이루어진다고 할 수 있다. 객체는 특정 데이터와 관련된 함수들을 가지는 프로그램
        모듈이라고 정의 할 수 있으며 실세계 영역에 있는 정보, 물건, 사물, 개체, 개념 등을 표현할 수 있다. 
        예를 들면 대학강의라고 정의된 클래스부터 현실세계에 속한 어떤 강의실에서 학생들의 성적평가 객체 또는 교수평가 객체 등을 생각할 수 있다. 
        객체는 속성(Attribute)과 행위(behavior)로 구분된다. 즉, 변수와 메소드로 이루어져있다.
         객체지향 프로그래밍에서 모든 객체는 추상화된다. 추상화란 객체가 자신의 정보를 내부에 감추고 있으면서 구체적인 것이 아닌 추상적인 내용만 외부에 알려주는 것을
         의미한다. 추상화는 정보은폐라고 표혆하기도 하고 연산자 추상화, 데이터 추상화 등이 있다. 
        자바스크립트 언어는 객체지향적 언어이기는 하지만 객체지향언어가 가지고 있는 자료의 추상화, 상속성, 다형성 의 특성을 지원하지 '않는'다. 
        그러므로 자바스크립트에서는 객체를 정의할 때 클래스나 구조체를 사용하지 않고 함수를 만들어서 객체를 생성한다. 
        따라서 객체정의를 위한 함수를 만들고 전달할 매개변수를 선언한 후 this를 사용하여 매개변수값을 전달한다. 자바스크립트 객체는 크게
        속성(property), 메소드(method), 이벤트(event)로 구성된다. 
        자바스크립트에서 다음 2가지 종류의 객체를 제공한다. 
         1) 사용자 정의 객체 : 사용자가 임의의 객체를 정의한 후 정의된 객체를 이용하여 객체를 생성하여 사용
         2) 내장 객체 : 미리 객체가 정의되어 있어서 사용자는 객체를 생성하여 사용

        
사용자 정의 객체

         자바스크립트에서 객체를 사용하기 위해서는 먼저 instance를 생성해야한다. 이것이 바로 객체의 정의이며, 이미 정의된 객체를 사용하기 위해서는 다음과 같은 생성자
         함수의 구조형식으로 이루어진다. 

         function 객체명(속성값1, 2, 3){
             this(객체자신).속성명1 = 속성값1;
             this(객체자신).속성명2 = 속성값2;
             this(객체자신).속성명3 = 속성값3;

             this.메소드명1 = 함수명1;
         }
         객체는 생성자 함수를 통하여 이루어지므로 먼저 생성자 함수를 만들어야 한다. 생성자 함수를 만드는 방법은 일반함수와 유사하지만 함수의 이름이 곧
         객체의 이름이 된다는 점과 메소드를 선언할 때 메소드로 사용하는 함수를 동일한 이름으로 지정해야 한다는 점이 조금 다르다.

객체 제어를 위한 for-in 문

        자바스크립트에서 객체를 효율적으로 이용하고자 for in 제어문을 사용하면 속성과 값을 활용할 수 있다. for in 문은 다음과 같은 형식으로 이루어진다. 
        for(k in obejct){
            st1; st2; stn;
        }
        k - 제어문의 변수 이름
        objectname = 객체이름
        변수 k에 대입되는 값은 객체의 속성들이 차례로 대입됨
        만약 객체내에 속성이 3개이면 속성의 갯수만큼 반복

javascript 에서 제공하는 내장 객체

        객체지향프로그램의 주된 장점중의 하나는 이미 만들어진 객체를 필요에 따라 사용할 수 있다는 점이며, 이를 내장객체라 한다. 
        내장객체는 자바스크립트 자체에서 지원하는 객체와 웹 브라우저 환경에서 지원하는 객체로 나눌 수 있다. 
         자바스크립트 자체에서 지원하는 객체는 String, Mate, Date, Array 객체와 eval, parseInt, parseFloat 등이 있고, 웹 브라우저 
         환경에서 지원하는 객체로는 parents, frame 등의 윈도우 객체와 흔히 사용하는 document, form 객체 등이 있다. 

         String : 문자열 처리에 관련된 내장 객체
         Date : 날짜 처리에 관련된 내장 객체
         Array : 배열의 처리에 관련된 내장 객체
         Math : 수학 연산 처리에 관련된 내장 객체
         Function : 간단한 함수를 정의하여 사용할 수 있도록 하는 내장 객체
         Screen : 서버로부터 웹페이지를 요구한 클라이언트 컴퓨터의 화면 해상도 등을 알아내는데 사용되는 내장객체
         Number : 문자열로 되어있는 숫자를 연산이 가능하도록 실제 숫자로 변환해주는 내장 객체
         Boolean : 불리언 값의 처리에 관련된 내장 객체

        Javascript에서 지원하는 내장객체들 중에서 String, Math 객체는 new 연산자로 객체를 생성하지 않고 바로 사용하는 정적객체(static obj)라
        하고, 정적객체는 자바스크립트가 실행되는 동안 자동으로 생성된다. 

        String 객체 
         string 객체는 문자열을 구성하는 문자들의 집합을 의미하고, 객체를 생성하는 new 연산을 사용하지 않으며, 문자열을 다양학 ㅔ처리할 수 있는 많은 
         메소드들과 속성을 제공한다.

String 객체의 각종 메소드

        anchor(문장) : 문장 위치로 페이지 이동
        big() : 기본 폰트보다 조금 더 큰 글자
        blink() : 깜빡이는 글자
        charAt(Index) : index 위치의 글자 확인
        bold() : 굵은 글씨체
        fixed() : 타자기체의 글자
        fontcolor(색) : 글자색 설정
        fonrsize(size) : 글자 크기 설정
        indexOf(문자열) : 문장 중에서 문자열의 인덱스 위치 검색
        italics() : 이탤릭체의 글자
        lastIndexOf(문자열) : 문장 중에서 문자열의 마지막 인덱스 위치 검색
        link(url) : 다른 위치나 사이트로 이동
        small() : 기본 폰트보다 조금 작은 글자
        strike() : 가운도 줄 그은 글자체
        split(대상, 기준문자) : 대상문자열ㅇㄹ 기존문자로 분리
        sub() : 아래첨자로 표현
        substring(index1, index2) : 1에서 2사이의 문자열을 문장중에서 검색
        sup() : 윗첨자로 표현
        toLowerCase() : 소문자로 변경
        toUpperCase() : 대문자로 변경
