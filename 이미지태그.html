이미지 삽입 태그 

        
inline이미지와 External 이미지

        웹 페이지에 접속하면 특별한 그래픽보기 프로그램을 실행하지 않았는데도 그래픽 이미지가 자동으로 나타나는 것을 많이 볼 수 있다.
         이처럼 html문서나 웹 페이지에 자연스럽게 조화되어 동작하는 이미지를 inline 이미지라 한다. 
         inline 이미지는 img태그를 이용해 만들 수 있다.

         inline 이미지는 웹 문서를 읽으면 바로 나타나는 이미지로, 모든 브라우저에서 지원하는 gif형식을 주로 사용한다.

         External 이미지는 웹 문서에서 필요하면 다운을 받아 사용할 수 있는 이미지로, 이미지를 압축하여 저장받아 다운받는 시간을 절약할 수 있는
         jpeg 형식을 많이 사용하게 된다.

         img 태그에서 src는 반드시 지정해야 하며, 삽입된 이미지에 대한 다양한 속성은 다음과 같다.

         src : 이미지의 url
         width : 너비지정, 생략하면 실제 넓이 적용되고, 너비만 주면 높이는 비례적으로 따라옴
         height : 높이 지정, ㅅ ㅐㅇ략하면 실제 높이 적용되고, 높이만 주면 너비는 비례적으로 따라옴
         align : 문자열과 함꼐 사용할 때 정렬 방법
         hspace : 이미지 양 옆으로 생성되는 여백, 픽셀 수로 지정
         vspace : 이미지 위 아래에 형성되는 여백, 픽셀 수로 지정
         border : 이미지의 테두리 선의 두꼐

이미지의 정렬

        웹 문서에서 텍스트와 이미지가 조화롭게 배열하기 위해 이미지 태그내의 얼라인 속성을 활용하여 적절하게 배치할 수 있다
        이미지와 텍스트의 정렬 : align = top, middle, bottom
        이미지의 위치 : align = left, right
        align=middle : 텍스트의 첫 줄은 이미지의 중간과 맞추고, 그다음 줄은 이미지의 다음 줄로 내려간다
        align=top : 텍스트의 첫 줄은 이미지의 상단과 맞추고, 그 다음 줄은 이미지의 다음 줄로 내려간다
        align= bottom : 텍스트의 첫줄은 이미지의 하단과 맞추고 그 다음 줄은 이미지의 다음 줄로 내려간다. align의 기본값은 bottom 이므로
        기본값을 설정하지 않으면 bottom으로 설정된다

        align=left : 이미지가 글자의 왼쪽에 와서 문단정렬이 된다.
        aling=right : 이미지가 글자의 오른쪽에 와서 문단정렬이 된다.

이미지 대체하기

        html에서 이미지가 제대로 전송되지 않았거나, 브라우저가 이미지를 받아들일 수 없을 때 대체하는 문구를 넣는 방법이 있는데, 이 방법은 alt속성을
        이용하여 실행할 수 있다. alt속성은 alt = "이미지설명" 의 형식을 사용한다.

위

        
이미지에 연결기능 추가하기

        이미지에 연결기능을 추가하는 방법은 텍스트에서와 마찬가지로 이미지 태그 바깥쪽에 앵커 태그를 씌워서 사용한다.
위 
        
이미지를 배경으로 사용하기

        이미지를 배경에 사용하려면 body태그에서 지정해야한다. 그리고 bgcolor를 이용하여 배경색을 지정하고 text를 이용하여 글자색을 결정한다.
