목록태그

 html에서 문서를 시각적으로 ㅉ아ㅣㅁ새있는 형태로 만들기 위해 목록 태그를 사용한다.
        목록태그는 번호없는 목록, 번호 있는 목록, 메뉴 목록, 디렉터리 목록, 정의 목록 등 다양한 형태가 제공되고 있으며
        목록을 만들기 위한 태그는 다음과 같은것이 있다.

        ul : 번호없는 목록(Unordered List)
        ol : 번호 있는 목록(Ordered List)
        menu : 메뉴 목록
        dir : 디렉터리 목록
        dl : 정의 목록

        다음은 목록 태그 하위에 사용하는 태그들이다.
        li : 리스트 항목
        lh : 리스트 제목
        dt : 정의 이름
        dd : 정의의 설명

        
번호없는 목록

        html에서 번호없는 목록은 각 항목 앞에 불렛(bullet)이 붙는 목록을 말하고 ul 태그를 이용하여 작성한다.
        번호없는 목록을 작성할 때 목록의 제목은 lh태그 안에 각 항목은 li태그 안에 작성해야 한다.
        li태그 하위에 ul태그를 넣어 계층적 정보를 나타낼 수 있다.
        bullet의 모양은 type 속성에 다음 값을 주어 지정할 수 있다.
        disc : 검은 원
        circle : 하얀 원
        square : 검은 사각형

번호 있는 목록

        html에서 번호있는 목록은 각 항목 앞에 번호가 붙는 목록으로 Ol태그를 사용한다.
        초기값으로 지정된 번호는 아라비아 숫자이며 번호 형식은 type 속성에 다음값을 주어 지정한다.

        i : 로마자
        a : 알파벳

    정의 목록
    
        정의목록은 어떤 항목에 대한 정의나 설명을 써줄 때 사용하며, dl태그를 사용한다. dl태그 내에서 lh태그는 목록의 제목을 dt태그 사이에서는 항목을
        dd태그 사이에서는 그 항목에 대한 정의나 설명을 쓴다.
        번호없는 목록이나, 번호 있는 목록과 마찬가지로 중첩 사용 가능하며, 하위 항목으로 들어감에 따라 문단이 조금씩 안쪽으로 들어간다.
