자바스크립트 문장 기초

 script 태그를 head 안에 삽입하는 이유는 미완성이거나 혹은 미처 해석되지 않은 자바스크립트 코드를 실행시킴으로써 발생하는 문제를
        미연에 방지하기 위해 삽입하고, body 태그 안에 들어가는 경우도 있는데 같은 이유이다.

        자바스크립트의 코드를 감추기 위해 저장된 파일을 불러와서 삽입하는 경우는 script태그의 src 속성을 사용한다.
        자바스크립트 파일의 확장자는 js 이어야 한다.

자바스크립트의 상수와 변수

        상수 : 상수는 그 값이 정해지면 변하지 않는 값을 말하며, 숫자(정수형, 실수형) 문자[열] 논리형, 널(null)값을 지원한다. 
        다음은 자료형을 정리했다.
        숫자 자료형 : 
            - 정수형 : 
                    - 10진법 : 소수점이 포함되지 않는 10진수
                    - 8진법 : 0으로 시작하는 숫자로 0에서 7까지의 조합으로 구성
                    - 16진법 : 0x로 시작하는 숫자로 0에서 F까지의 숫자, 문자 조합으로 구성
            - 실수형 :
                    - 일반 실수형 : 소수점을 포함하는 10진수
                    - 지수형태의 실수형 : 일반 실수형에서 소수점 이하 자리수가 많은 경우 지수형으로 간단히 표기하기 위해 사용
                    
        문자열 : 이중따옴표 내에 있는 연속된 문자들을 문자열이라고 부름. 영문자, 숫자, 특수문자 등 키보드로 입력할 수 있는 모든 문자 포함, 이중따옴표 
                대신 단일따옴표 사용 가능
        논리형 : 논리타입은 참과 거짓의 두 값들 중 하나만을 가짐
        null : 변수의 값이 정의되지 않아서 변수에 아무 값도 들어있지 않다는 의미
        NaN : not a number 는 숫자가아니다 라는 것을 의미. 예로 0/10이 있음

        변수 : 상수 값을 저장하기 위한 기억장소의 이름으로 프로그램이 실행되면서 변수(기억장소)에 기억된 내용의 변경이 가능하고,
            자바스크립트에서 변수의 분류는 지역변수와 전역변수로 구분되며 차이점은 다음 표와 같다.
        
         전역변수 :
            1) 함수 밖에서 선언된 모든 변수
            2) 변수이름 앞에 var 키워드를 붙여도 되고 붙이지 않아도 된다
            3) 함수의 외부 또는 내부에서 참조가 가능하다
        
        지역변수 :
            1) 함수 내에서 선언된 모든 변수
            2) 보통 변수 이름앞에 var키워드를 붙여준다
            3) 변수가 선언되어 있는 함수의 내부에서만 참조할 수 있다.
        *주의* : 변수를 선얺할 때 var키워드가 있는 변수는 초기값을 할당해도 되고 하지 않아도 되지만, var키워드가 없는 변수는 반드시 초기 값이
                주어져야 한다.
        
        변수의 특징 :
            1) 영문자나 _시작하여 둘째 문자부터는 영문자와 숫자를 혼용할 수 있다.
            2) 대입되는 자료형에 따라 자동으로 그 형이 결정된다.
