자바스크립트의 연산자와 제어구조

        자바스크립트에서는 산술, 논리, 비트 및 할당 연산자를 제공하며 기타 연산자들도 제공한다. 
        연산자들은 자바스크립트에서 식을 만들거나 문장을 작성할 때 필요하다. 
        자바스크립트가 제공하는 연산자는 컴파일러에 산술 또는 논리연산을 수행하도록 지시하는 기호이다. 
        이러한 연산자를 자바스크립트에서는 매우 다양한 형태로 지원하고 있다. 

        
연산자의 종류


        연산자는 컴파일러에 산술 또는 노리연산을 수행하도록 지시하는 기호이다.
        그리고 이러한 연산자의 작용을 받는 것을 피연산자라 한다. 

        부호 연산자 : +, -
        증감 연산자 : ++, --
        산술 연산자 : + - * / %
        논리 연산자 : && , ||
        관계 연산자 : ! < <= > >= !=
        대입 연산자 : = += -= *= /= %=
        조건 연산자 : ? :
        배열 연산자 : []
        메소드 연산자 : ()
        비트 연산자 : &, |
        캐스트 연산자 : (데이터형)

        
증가연산자와 감소연산자

        증가연산자(++)과 감소연산자(--)에서 중요한 것은 전위형, 후위형인데 전위형(++n)은 자신의 값을 증가시키고 그 값을 사용하나 후위형은 사용 한 후 증감시킨다. 

        
비트연산자


        연산자      이름        유형        사용형식
         &       비트AND      이항연산자    x & y
         |       비트or       이항연산자    x | y
         ^       비트 XOR     이항연산자    x ^ y
         ~       비트 not     단항연산자    ~x

         비트AND 연산자 : 두 개의 비트가 모두 1일 경우에만 결과 값이 1이 된다. 
         비트OR 연산자 : 두 개의 비트 중 하나만 1이여도 결과 값이 1이 된다. 
         비트XOR 연산자 : 두 개의 비트가 서로 다를 경우에만 결과 값이 1이 된다. 
         비트NOT 보수 연산자 : 이 연산자는 해당 비트를 toggle 시키는 역할을 한다. 

        
시프트 연산자

        시프트 연산자는 우변에 있는 피연산자에 의해 주어지는 자릿수만큼 피연산자의 비트를 좌측 또는 우측으로 자리이동을 하는 연산자이며 
        이때 좌측 또는 우측으로 밀려나가는 비트는 모두 버리게 된다. 

        다음은 자바스크립트에서 제공하는 시프트 연산자의 종류를 나타낸다

        연산자    이름                   유형        사용형식
        <<      왼쪽시프트              이항연산자     x << y
        >>      오른쪽시프트            이항연산자     x >> y
        >>>     0으로채우기 오른쪽 시프트  이항연산자     x >>> y

        왼쪽시프트 : 왼쪽 시프트 연산자는 x << y에서 x를 y에서 지정한 비트만큼 왼쪽으로 이동시키는 연산자이다. 비트를 왼쪽으로 이동시키면 오른쪽에
                빈자리가 생기는데 이 자리는 모두 0으로 채운다.
        오른쪽 시프트 : 오른쪽시프트 연산자는 왼쪽 시프트 연산자와 마찬가지로 x>>y에서 x를 y에서 지정한만큼 오른쪽으로 이동시키는 연산자이다. 
            비트를 오른쪽으로 이동시키면 왼쪽에 빈자리가 생기는데 이 자리는 왼쪽쉬프트랑은 달리 부호비트에 따라 0또는 1로 채운다. 
            즉 부호비트가 0이면 빈자리를 0으로 채우고 1이면 빈자리를 1로 채운다. 
        
        0으로 채우기 오른쪽 시프트 : 이 연산자는 오른쪽 시프트 연산자와 동일하게 동작한다. 단, 오른쪽으로 이동한 후 새로 생기는 왼쪽의 빈자리는 모두 0으로 채운다.
