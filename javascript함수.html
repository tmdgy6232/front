자바스크립트의 함수

        함수는 하나의 모듈로서 참조되고 실행될 수 있는 문장들으 ㅣ블록을 지정하는 것으로 독립적으로 처리할 수 있는 작업을 기술하는 프로그램의 단위이다. 
        함수는 하나의 이름으로 여러 개의 연산을 결합시킬 수 있으며, 코드를 간결하게 작성할 수 있는 장점을 가지고 있다. 

        자바스크립트에서의 함수는 일반적으로 특정값을 리턴받기 위해 사용되며, 값을 리턴하지 않는 함수들은 대게 변수나 객체를 수정하기 위해 사용된다. 

        또한 함수 이름 뒤에 오는 괄호 안에 인수라고 하는 정보를 넣어 다른 함수에 정보를 전달할 수 있다. 함수에 전달되는 정보를 인수 또는 매개변수라고 한다. 
        함수 중에는 인수를 선언하지 않고 사용하는 것도 있고, 하나 이상의 인수를 선언하여 사용하는 것도 있다. 자바스크립트에서 함수는 사용자 정의 함수와 내장
        함수로 구분되며 전자는 사용자가 직접 정의하여 사용하는 함수이고, 후자는 자바스크립트에서 제공하는 함수이다. 

        
 사용자 정의 함수

        - 함수의 정의 및 호출
         : 자바스크립트에서는 내장된 웹 브라우저 객체가 제공하는 많은 함수들이 있지만, 사용자가 필요로 하는 함수를 모두 제공할 수는 없다. 
         이럴 때 사용자는 자신이 필요로 하는 함수를 정의하여 사용할 수 있는데 이를 사용자 정의 함수라 한다. 
          사용자 정의 함수는 먼저 함수를 정의해두고 이를 호출하여 사용하여야 하는데, 함수 정의는 함수명과 자바스크립트 문장으로 구성된다. 
          자바스크립트에서 사용자 정의함수는 function으로 시작하며 문장들 사이에 함수의 시작과 끝을 구분하기 위하여 중괄호를 사용하여 구분한다. 
          함수를 정의하는 위치는 head 태그 사이에 정의되고 리턴값의 형과 매개변수의 형은 지정하지 않아도 된다. 
          사용자 정의 함수에서 함수의 정의 구조는 다음과 같은 형식을 따른다. 
           function 함수명(매개변수){
               내용
           }
        그리고 사용자 정의함수는 다음과 같은 형식으로 함수 호출이 이루어진다
        함수명(매개변수);

        
 지역변수와 전역변수의 차이 

        자바스크립트에서 변수를 설정할 때 일반적으로 var 명령을 이용하여 변수를 선언하는데, 이 변수는 함수 안에서만 적용 가능하고 함수를 벗어나면 그 변수는
        적용되지 않는다. 이러한 변수를 지역변수라 한다. 
         이와는 반대로 var명령을 사용하지 않고 변수를 직접 선언할 수 있는데 이 변수는 문장 전체에 변수의 내용이 적용되는 기능을 가지고 있다. 
         이러한 변수를 전역변수라 한다. 
         형식
         function 함수명(){
             var 변수명; // 지역변수
             변수명;    //전역변수
         }
         
 내장함수 

         내장함수는 자바스크립트 자체에서 지원하는 함수를 읨히ㅏ며, 독립적으로 객체의 참조없이 어느 위치에서든지 혹은 어떤 객체에서도 호출하여 사용할 수 
         있는 함수를 말한다. 자바스크립트에서는 제공하는 내장함수는 객체에 의존하지 않기 때문에 자바스크립트 구문 안에서 자유롭게 사용할 수 있으며, 함수의
         종류에는 문자열과 수치연산을 위한 것들이 있다. 
         다음은 자바스크립트 내장함수와 간략한 설명을 한 것이다. 
         alert(String) : 확인버튼이 있는 알림 대화상자를 생성해주는 함수
         cofirm() : 확인과 취소 버튼이 있는 확인/취소 대화상자를 생성해주는 함수
         prompt() : 확인과 취소 버튼이 있고 입력상자가 있는 대화상자를 생성해주는 함수
         parseint(문자열, [진수]) : 문자열을 정수형으로 변환하여 리턴한다, 진수는 8, 10, 16진수 중에서 하나를 설정할 수 있으며, 기본값은 10진수이다. 
         parsefloadt(문자열) : 문자열을 실수값으로 변환하여 리턴한다. 만약 첫번째 글자를 숫자로 변환하지 못할 경우에는 NaN으로 리턴한다. 
         isNaN(검색할숫자) : 제공된 ㅜㅅ자가 타당하지 않ㅎ은 연산결과가 발생할 경우에는 true, 아니면 false를 리턴한다. 
         isFinite(검색할 숫자) : 제공된 숫자가 양의 무한대나 음의 무한대를 가지는 경우에는 false, 아닐때는 true를 리턴함
         escape(특정문자) : A-Z, a-z, 1-0, @*-_+./를 제외한 모든 문자를 %16진수의 아스키 코드값으로 변환한다. 1바이트 문자는 %xx의 형태로
                        2 바이트의 문자는 %uXXXX의 형태로 변환한다. 
        unescape(아스키코드) : %16진수의 아스키코드를 문자로 변환한다. 
        eval(수식문자열) : 문자열을 숫자로 변환한 후 그 결과값을 리턴한다.
